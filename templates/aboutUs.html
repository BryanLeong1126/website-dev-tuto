About us

Our traffic monitoring system uses embedded machine learning with Jetson Nano and Yolo V8 to monitor traffic and detect congestion. The Jetson Nano is a small, powerful computer designed to power entry-level edge AI applications and devices. Yolo V8 is a state-of-the-art object detection and image segmentation model created by Ultralytics. By combining the power of the Jetson Nano with the advanced capabilities of Yolo V8, our system can accurately monitor traffic in real-time and quickly detect congestion, allowing for more efficient traffic management. This system is an example of embedded machine learning, where machine learning algorithms are deployed to run on microcontrollers.
